# MysteryDice - Daily Mystery Cases

## App Info
- **App Name**: MysteryDice
- **Bundle ID**: com.ggsheng.MysteryDice
- **Category**: Games / Puzzle / Adventure
- **Target**: iOS 17.0+
- **Market**: Global (English), focus on Europe & America

## Core Concept
每天一个新案件(谋杀/盗窃/失踪)，玩家决定怎么破案。每一步选择影响结局，有解密/破解元素，多结局可重玩。
